ARInCodeExperimentDetails Class Reference

Inherits from ARExperimentDetails : NSObject
Declared in ARInCodeExperimentDetails.h

Overview

An object containing useful information about an in-code experiment These are retrieved via the ARExperimentManager#getInCodeExperimentDetails method.

If the experiment is not running then some fields will be nil, including currentVariantId, experimentId, experimentName, experimentType. You can check whether the experiment is currently running with isRunning.
