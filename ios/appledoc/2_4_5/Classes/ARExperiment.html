ARExperiment Class Reference

Inherits from NSObject
Declared in ARExperiment.h

Overview

Represents an in-code experiment, including its name and variants.

Instead of creating ARExperiment objects directly, it’s better to use the methods defined in ARExperimentManager.

addVariant:isDefault:

Add a variant to the experiment.

- (void)addVariant:(NSString *)variantName isDefault:(BOOL)isDefault

Parameters

variantName

The name of the variant you’re adding

isDefault

Whether this is the default variant for this experiment. Note that if this is the first variant for the test, it is assumed to be the default, regardless of the value of isDefault, until another variant is added with isDefault set to YES.

Discussion

Add a variant to the experiment.

Warning: Attempting to add two variants with the same name will result in the second being ignored.
